# colab_codes

A repository for practice code and learning materials intended for use in Google Colab.

## Repository Structure

```
colab_codes/
├── notebooks/          ← .ipynb files (Colab notebooks)
│   ├── linear_algebra/
│   ├── graph_theory/
│   └── algorithms/
├── scripts/            ← .py utility scripts (helper functions, etc.)
├── data/               ← small datasets if needed
└── README.md
```

### Directories

- **notebooks/**: Contains Jupyter notebooks for Google Colab organized by topic
  - `linear_algebra/`: Linear algebra problems and solutions
  - `graph_theory/`: Graph theory concepts and implementations
  - `algorithms/`: Algorithm implementations and practice problems
- **scripts/**: Reusable Python utility scripts and helper functions
- **data/**: Small datasets used by the notebooks

## Usage

1. Upload notebooks to Google Colab to run and experiment
2. Use scripts for helper functions and utilities
3. Organize code by topic in the appropriate notebook subdirectory
