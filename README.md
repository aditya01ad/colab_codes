# colab_codes 🧠

Personal coding practice repo — used with Google Colab.

## Structure
```
notebooks/
├── linear_algebra/    ← eigenvalues, matrix decompositions, etc.
├── graph_theory/      ← spectral theory, adjacency matrices, etc.
└── algorithms/        ← sorting, searching, dynamic programming, etc.
scripts/               ← reusable Python utility functions
data/                  ← small datasets
requirements.txt       ← dependencies
```

## Workflow
1. Open notebook via `File → Open notebook → GitHub` in Colab
2. Work, experiment, break things
3. Save back: `File → Save a copy in GitHub`

## Topics
- Linear Algebra (NumPy, SciPy, SymPy)
- Graph Theory (NetworkX, spectral methods)
- Algorithms & Data Structures
- Numerical Methods
